# v0.5.3 Release Notes

Release date: February 28, 2026

## What Changed
- Added `e2e-runner` workflow to reset + smoke in one command
- Added Makefile targets `verify` and `verify-up`
- Added ops troubleshooting guide for E2E failures

## Commands
```bash
make verify
make verify-up
```

Manual runner:
```bash
pnpm --filter ideaapp-backend e2e:verify
```

## Known Limitations
- Runner assumes API is reachable at `API_BASE_URL` (default http://localhost:7101)
- Docker log tailing only works if docker compose is installed
