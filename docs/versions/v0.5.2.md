# v0.5.2 Release Notes

Release date: February 28, 2026

## What Changed
- Added Makefile helpers for E2E smoke workflows
- Added safe E2E reset script for synthetic data cleanup
- Documented Docker + E2E commands

## Commands
```bash
make up
make smoke-up
make smoke
make reset
```

Override API URL:
```bash
API_BASE_URL=http://localhost:7101 make smoke
```

## Known Limitations
- Reset script only deletes synthetic E2E records (by email tag)
- Hard reset is not enabled by default
